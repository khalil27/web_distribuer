# Skill Rating Service

## Project Overview
The **Skill Rating Service** is a dynamic web application that allows users to rate various skills and provide valuable feedback. Developed using **Spring Boot**, this service enhances user engagement by offering skill assessments, real-time notifications, and automated email alerts, facilitating skill development and progress tracking.

## Features

- **Skill Rating**: Rate different skills and leave insightful comments.
- **Real-Time Notifications**: Receive instant updates on new ratings and comments.
- **Email Alerts**: Get automated email notifications when a new rating or comment is made.

## Technologies Used

- **Spring Boot**: Backend framework for building scalable web applications.
- **Spring Data JPA**: For ORM support and seamless database interactions.
- **Spring Security**: Ensures secure authentication and authorization.
- **Thymeleaf**: Template engine for dynamic content rendering.
- **JavaMailSender**: For sending email notifications.
- **Docker**: For containerizing the application for consistent deployment.

## Prerequisites

Ensure you have the following installed:
- **Java 11 or higher**: Required for running the application.
- **Maven**: For building the project and managing dependencies.
- **Docker** (optional): For running the application in a containerized environment.
